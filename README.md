# HireTwin AI - Your AI Career Operating System

**"From Zero Experience to Dream Job — Powered by AI"**

## 🚀 Overview

HireTwin AI is a complete, production-ready AI-powered career platform that helps users create their career identity from scratch and automatically generates everything needed to land their dream job.

## ✨ Features

### Complete User Journey (17+ Screens)

1. **Splash Screen** - Eye-catching landing with animated AI branding
2. **Onboarding** - 3-step interactive introduction to platform features
3. **Authentication** - Modern login/signup with Google, GitHub, LinkedIn integration
4. **AI Career Questionnaire** - ChatGPT-style interface that builds your profile
5. **Main Dashboard** - Premium card-based navigation with career metrics
6. **Resume Generator** - Multi-section form builder with dynamic fields
7. **Resume Preview** - Professional resume layout with ATS score visualization
8. **ATS Analyzer** - Keyword analysis with before/after improvement examples
9. **Career Prediction** - AI-powered role matching with probability scores
10. **Skill Gap Analyzer** - Current vs required skills comparison
11. **Learning Roadmap** - 30/90/180-day personalized learning plans
12. **Portfolio Generator** - Full website preview with modern design
13. **LinkedIn Generator** - Optimized profile content with copy-to-clipboard
14. **Interview Preparation** - Mock interviews with performance analytics
15. **Career Growth Dashboard** - Comprehensive analytics with charts and graphs
16. **Premium Subscription** - Beautiful pricing page with 3 tiers
17. **404 Page** - Stylish error page
